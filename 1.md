# 在虚拟机里下载安装部署openclaw

下载思路：
第一步下载虚拟机，这里我使用的是VirtualBox，
第二步下载unburnt，这里要在中科大的镜像里下载，因为当时别的几家我都试了，下载不了到你合适的。
下载好了之后，请做个快照，保存一下现在的进度。
在下载openclaw之前，先想办法，打开双向剪切板，这样方便后面的操作，一定要，在你的主机上复制好一段文字可以在你的虚拟机上可以粘贴，一定要完成这一步之后，再下载openclaw，这样你下载的速度会快很多。
第三步下载openclaw,下载之前先创建一下你的API，这里我使用的是minimax2.5大模型，当然这一步，也是你会遇到许多问题的一步，不要担心，遇到问题了，记得截图发给deepseek或者Google的模型看它给的结果。
下面是详细步骤
   
### 1第一步：在Windows上安装Oracle VM VirtualBox
下载VirtualBox：访问VirtualBox官方网站的下载页面，选择“Windows hosts”版本下载安装包 。
运行安装程序：双击下载的 .exe 文件，启动安装向导。通常只需一路点击“下一步” 。
网络警告：安装过程中，系统可能会弹出“网络接口”警告，提示将重置网络并安装虚拟网卡驱动程序，请点击“安装”继续。
完成安装：安装成功后，继续安装下载虚拟机。

## 2第二步：下载虚拟机

这里我们要做中科大的镜像网站里下载这个Ubuntu，
这一步是关键，我们直接从你指定的中科大镜像站获取Ubuntu安装文件。
访问镜像站：在Windows电脑的浏览器中，打开中科大开源软件镜像站的Ubuntu Releases页面：http://mirrors.ustc.edu.cn/ubuntu-releases/ 。
进入版本目录：点击你想要的Ubuntu版本对应的文件夹。建议选择 22.04/ 或 24.04/ 这样的长期支持版本，更稳定。
找到并下载ISO文件：进入版本文件夹后，找到文件名类似于 ubuntu-22.04.5-desktop-amd64.iso 的文件。
desktop 表示桌面版（带图形界面），是我们需要的。
amd64 表示是给64位电脑用的。点击这个文件，浏览器就会开始下载。把它保存到一个你记得住的地方，比如D盘的Downloads文件夹。

### 3第三步：创建虚拟机虚拟机并安装Ubuntu

新建虚拟机：

打开VirtualBox，点击工具栏上的“新建”按钮 。
名称：填写 "Ubuntu-OpenClaw"。
ISO镜像：点击下拉菜单，选择“选择或输入光盘映像文件”，然后找到并选中你刚刚从中科大镜像站下载的Ubuntu ISO文件 -9。
类型：系统会自动识别为“Linux”和“Ubuntu (64-bit)”。
点击“下一步”。
分配硬件资源：
内存：建议分配 4096 MB (4GB) 或更多（不要超过你电脑物理内存的一半）。
处理器：根据你的CPU核心数，分配2个或更多核心。
硬盘：选择“现在创建虚拟硬盘”，然后点击“创建”。虚拟硬盘大小建议分配 50GB以上。
安装Ubuntu系统：
在VirtualBox主界面选中你刚创建的虚拟机，点击“启动”。
虚拟机将自动从你挂载的ISO镜像启动，进入Ubuntu安装界面。按照屏幕提示选择语言、时区、键盘布局，
并创建你的用户名和密码（这里建议你的密码可以使用123456，这样方便使用，因为在虚拟机里输入密码是它是不会显示的）。
关键一步： 当提示“安装类型”时，选择“擦除磁盘并安装Ubuntu”（这只会清除虚拟硬盘，对真实电脑无影响）。
继续直到安装完成。系统提示重启时，如果卡在重启界面，可以手动关闭虚拟机窗口，选择“关机”，然后再重新启动。
这是建议你做个快照，保存一下进度（命名为Ubuntu安装完成）。

###  第四步：将Ubuntu软件源更换为中科大(USTC)镜像源
现在需要你做的是，打开双向复制粘贴，这时还需要你在下载一些东西，这时候你需要问一下deepseek，按它的操作进行，
这样方便后面的操作，
一定要，在你的主机上复制好一段文字可以在你的虚拟机上可以粘贴，一定要完成这一步之后，
再下载openclaw，这样你下载的速度会快很多。
系统装好后，第一件事就是把下载软件的源头换成国内的，这样后面安装OpenClaw时会快很多。
打开终端：在Ubuntu桌面，按 Ctrl + Alt + T 快捷键打开终端。
配置国内源头，建议你去问一下deepseek，按deepseek的提示进行，这样方便后面的操作，
备份原始源列表（一个好习惯）：
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
编辑源列表文件：
sudo nano /etc/apt/sources.list
替换内容：

这个文件里原本是以 http://archive.ubuntu.com/ 开头的地址。

用下面的内容全部替换掉文件中的原有内容。请注意：如果你的Ubuntu版本是22.04 (Jammy)，直接复制下面内容。如果是24.04，请将下面所有 jammy 替换为 noble 。
deb https://mirrors.ustc.edu.cn/ubuntu/ jammy main restricted universe multiverse
deb https://mirrors.ustc.edu.cn/ubuntu/ jammy-updates main restricted universe multiverse
deb https://mirrors.ustc.edu.cn/ubuntu/ jammy-backports main restricted universe multiverse
deb https://mirrors.ustc.edu.cn/ubuntu/ jammy-security main restricted universe multiverse
保存并退出：按 Ctrl + O 保存文件，然后按 Ctrl + X 退出nano编辑器 。

更新软件源：让系统应用新的镜像源配置。
sudo apt update
看到下载速度有明显提升，就说明配置成功了。

### 第五步：在Ubuntu中安装、初始化并启动OpenClaw

安装Node.js：OpenClaw需要Node.js 22或更高版本 。在终端中执行以下命令：

bash
# 使用NodeSource官方源安装Node.js 22
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs
# 配置npm国内镜像，加速后续安装 [citation:4]
npm config set registry https://registry.npmmirror.com
验证安装：
node -v
npm -v
如果显示的版本号符合要求，则说明安装成功。

全局安装OpenClaw：
sudo npm install -g openclaw@latest
或者使用这个命令npm install -g openclaw@latest --verbose，这个可以看到安装的进度
//这时，在下载时可能时间较长，请耐心等待，下载时可能还会遇到问题请及时联系deepseek。
验证OpenClaw安装：
openclaw --version
启动初始化向导：这是配置OpenClaw的关键步骤，它会引导你完成设置 。
openclaw onboard --install-daemon
根据屏幕提示进行操作，你需要准备好一个大语言模型的 API Key（建议你使用比如minimax2.5等），或者你也可以先选择“Custom Provider”跳过，后续再配置 。

检查运行状态：

openclaw status
生成访问令牌并启动Web控制台：

# 生成令牌
openclaw token generate

# 启动Web控制台，命令执行后会自动在浏览器中打开管理界面 
openclaw dashboard
如果浏览器没有自动打开，你可以手动在Ubuntu的浏览器中输入 http://localhost:18789/?token=，并将上一步生成的令牌粘贴在后面。
打开后就可以交流了，但是你需要给你的minimax2.5买点token,一般开始你买1块钱它会送你15块钱，这样你的AI才能使用你的账号的模型。
恭喜！至此，你已经成功地在Windows电脑的虚拟机中，完全通过你指定的中科大镜像源，搭建起了属于自己的OpenClaw个人AI助理。
